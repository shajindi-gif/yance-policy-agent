### 展示技巧

1. **开场（前30秒）**
   - 站定后再开口，不要边走边说
   - 第一句话要有力量："各位评委好，我是...，今天我要分享的是..."
   - 眼神扫视全场评委，建立连接

2. **中段（30秒-5分钟）**
   - Demo 环节是重点，确保操作流畅
   - 如果Demo加载时间超过5秒，用语言填充
   - 每切换一页，先说标题再展开

3. **收尾（最后1分钟）**
   - 语速放慢，增强感染力
   - 最后一句话要清晰有力，留3秒静默
   - "谢谢大家！"后等待1-2秒再移动

### 时间控制

- 每页切换时瞄一眼计时器
- 如果超前：在Demo环节多展示一些细节
- 如果超时：压缩团队和需要支持页面（这两页可以口头带过）
- **绝对不能超时**——超时会打断评委节奏，影响印象

## 评委问答环节（3分钟）

### 应对策略

| 场景 | 应对 |
|------|------|
| 熟悉的问题 | 简洁回答，3-4句话，不啰嗦 |
| 不熟悉的问题 | 诚实说"这个问题很好，我们目前的思考是..." |
| 尖锐质疑 | 先认同再解释："您说的有道理，我们的做法是..." |
| 超出范围的问题 | 承认边界："这确实超出了我们目前的覆盖范围，我们会认真考虑" |
| 涉及金融/承诺的问题 | 明确边界："我们不涉及金融投资，不承诺申请结果" |

### 高频问题准备（Top 5）

1. **"你们和直接用ChatGPT问政策有什么区别？"**
   > "三个核心区别：第一，我们基于真实的政策知识库，不会出现幻觉；第二，我们结合企业画像做精准匹配，而不是泛泛回答；第三，我们嵌入园区工作流，能生成报告和工单，不只是聊天。"

2. **"你的壁垒是什么？大公司做怎么办？"**
   > "我们的壁垒在于政策知识工程的深度和园区场景的理解。大公司做通用工具，我们做垂直场景。坦率说作为初创公司护城河还浅，但我们靠执行速度领先。"

3. **"谁付费？怎么收费？"**
   > "园区公司付费，SaaS订阅3000-20000元/月，按园区规模分级。园区公司有明确的企业服务预算，政策服务是其中的刚需部分。"

4. **"当前做到什么阶段了？"**
   > "原型验证阶段。核心Demo已完成，可以现场演示。正在推进徐汇区的园区试点。坦率说产品还在早期，需要更多真实场景打磨。"

5. **"是否涉及金融投资领域？"**
   > "不涉及。我们专注产业政策服务，帮助企业了解政府出台的产业政策优惠。与金融投资、证券研究有明确边界，在产品文档中也有明确声明。"

### 问答礼仪

- 认真听完问题再回答，不打断评委
- 回答时眼神看向提问者，适当兼顾其他评委
- 不确定的问题不硬答，诚实比什么都重要
- 回答控制在3-5句话，不要展开成长篇大论
- 如果3分钟快用完了，简洁收尾："感谢各位评委的时间，期待后续交流。"

## 应急预案

| 场景 | 应急方案 |
|------|---------|
| Demo系统崩溃 | 切换到提前录制的Demo视频，口头讲解 |
| 投影/屏幕故障 | 用口头描述替代视觉展示，重点讲核心逻辑 |
| 网络中断 | 使用手机热点；Demo使用本地缓存数据 |
| 翻页笔失灵 | 用笔记本键盘翻页，或请助手协助 |
| 演讲超时 | 跳过团队和需要支持页面，口头带过 |
| 紧张忘词 | 停顿2-3秒，看PPT提示，再继续。不要道歉 |
| 评委尖锐质疑 | 先认同再解释，保持冷静和专业 |

## 心态提醒

> - 你不是在做报告，你是在和一个感兴趣的听众分享你在做的事情
> - 诚实比完美更重要——评委能感知到真诚
> - 评委提问不是刁难，是帮助你想得更清楚
> - 7分钟很短，说重点、说人话、说真话
