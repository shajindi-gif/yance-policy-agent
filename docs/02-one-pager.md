# YanCe Policy Agent - 一页纸概览

> **上海衍策引擎人工智能科技有限公司** | yance.ai

---

## 一句话定位

**YanCe Policy Agent 是园区公司的 AI 政策服务 Agent，让惠企政策精准触达每一家入驻企业。**

---

## 问题 Problem

- 园区涉及国家/市/区三级政策数百条，更新频繁，人工难以全面掌握
- 一个园区几十到几百家企业，行业、阶段、需求各异，无法逐一匹配
- 政策查询、匹配、申报、跟踪分散在不同系统和人员手中，服务无法闭环

---

## 解决方案 Solution

一个 AI Agent 系统，整合**政策库 + 企业画像 + 智能匹配 + 材料生成 + 服务工单**，为园区公司提供端到端的政策服务能力。

---

## 产品 Product

| 模块 | 功能 |
|------|------|
| PolicyCopilot | 自然语言政策问答 Agent |
| CompanyProfile | 企业多维画像引擎 |
| SubsidyMatch | 政策智能匹配引擎 |
| BriefMaker | 申报材料和服务报告生成器 |
| ParkOps | 园区政策服务运营工作台 |

---

## 市场 Market

- **目标客户**：园区公司 / 产业园区运营方 / 孵化器 / 众创空间 / 招商运营主体 / 政企服务公司
- **市场切入**：上海徐汇区，漕河泾开发区及周边园区生态
- **市场规模**：全国产业园区数以万计，仅上海就有数百家各类园区和孵化器
