## 产品定位

YanCe Policy Agent 是一款面向**园区公司**的 AI 政策服务 Agent。

它为产业园区运营方、孵化器、众创空间、招商运营主体和政企服务公司提供智能化的政策服务能力，包括：

- **政策匹配**：将园区政策与入驻企业进行精准匹配
- **企业画像**：构建入驻企业的多维画像，识别政策适用性
- **材料生成**：自动生成政策申报材料清单和辅助文档
- **服务工单**：将政策服务流程转化为可追踪的工单体系
- **园区运营**：为园区管理方提供政策服务运营工作台

## 核心价值主张

> 让每一家园区公司都拥有一个永不休息的政策服务 Agent，让每一项惠企政策都能精准触达适用企业。

## 产品模块

| 模块名称 | 功能描述 |
|---------|---------|
| PolicyCopilot | 政策问答 Agent，支持自然语言政策咨询 |
| CompanyProfile | 企业画像引擎，构建企业多维特征 |
| SubsidyMatch | 政策匹配引擎，精准匹配企业与政策 |
| BriefMaker | 材料生成器，生成申报材料和服务报告 |
| ParkOps | 园区运营工作台，管理政策服务全流程 |

## 技术特点

- **RAG（检索增强生成）**：基于政策知识库的精准检索，确保回答有据可依
- **可追溯**：每一条政策推荐都可以追溯到原始政策文件
- **可解释**：匹配结果附带详细的匹配逻辑和依据
- **可复核**：支持人工复核环节，确保政策推荐的准确性
- **不承诺结果**：系统不承诺政策申请成功或具体补贴金额

## 重要声明

**YanCe Policy Agent 不涉及以下领域：**
- 金融投资咨询
- 股票、基金、ETF、证券研究
- 加密货币交易或建议
- 任何形式的投资收益承诺

**YanCe Policy Agent 始终坚持：**
- 政策推荐可追溯、可解释、可复核
- 不承诺政策申请成功
- 不承诺具体补贴金额
- 尊重企业数据隐私
- 合规运营
