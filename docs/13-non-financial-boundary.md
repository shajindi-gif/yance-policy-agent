# 非金融业务边界声明 - YanCe Policy Agent

> 明确本项目的业务边界，排除金融投资相关活动

---

## 声明目的

本文档旨在明确 **YanCe Policy Agent（衍策政策Agent）** 的业务边界，清晰界定本项目**不涉及**和**不涉及**的领域，防止产品定位被误解为金融投资相关服务。

---

## 本项目明确不涉及以下领域

### 1. 证券研究与投资建议

| 不涉及内容 | 说明 |
|-----------|------|
| 股票研究 | 不提供任何个股分析、目标价预测或买卖建议 |
| 基金研究 | 不提供基金评价、基金排名或基金投资建议 |
| ETF 分析 | 不提供 ETF 板块轮动、ETF 投资建议 |
| 证券研究报告 | 不生成任何形式的证券研究报告或投资备忘录 |

### 2. 交易与投资管理

| 不涉及内容 | 说明 |
|-----------|------|
| 股票交易 | 不提供股票交易服务或交易信号 |
| 基金管理 | 不管理任何投资组合或基金 |
| 交易策略 | 不设计或推荐任何交易策略 |
| 收益承诺 | 不承诺任何投资收益或回报率 |

### 3. 加密货币与数字资产

| 不涉及内容 | 说明 |
|-----------|------|
| 加密货币 | 不涉及任何加密货币交易、咨询或推荐 |
| NFT | 不涉及 NFT 投资或交易 |
| DeFi | 不涉及去中心化金融协议或产品 |
| 数字资产 | 不涉及任何形式的数字资产投资建议 |

### 4. 金融咨询与理财

| 不涉及内容 | 说明 |
|-----------|------|
| 理财规划 | 不提供个人或企业理财规划服务 |
| 资产配置 | 不提供资产配置建议 |
| 风险管理 | 不提供金融风险管理咨询 |
| 税务筹划 | 不提供个人/企业税务筹划建议（仅提供产业政策信息） |

## 本项目的业务范围

### YanCe Policy Agent 专注于以下领域

| 业务领域 | 具体内容 |
|---------|---------|
| **产业政策服务** | 帮助企业了解和申请政府出台的产业政策优惠 |
| **园区运营服务** | 为园区公司提供政策服务工具和运营支持 |
| **企业画像** | 构建企业多维特征画像，用于政策匹配 |
| **政策匹配** | 基于企业画像匹配适用政策，生成匹配报告 |
| **材料辅助** | 生成申报材料清单和政策服务报告 |
| **服务工单** | 管理园区政策服务全流程 |

### 数据范围

| 数据类型 | 来源 | 用途 |
|---------|------|------|
| 政策文件 | 政府官方网站公开发布 | 政策查询和匹配 |
| 企业信息 | 企业提供 + 公开信息 | 构建企业画像 |
| 行业数据 | 公开统计数据 | 行业分析和趋势参考 |

### 不包含的数据

| 数据类型 | 说明 |
|---------|------|
| 证券行情数据 | 不接入、不展示、不分析任何股票/基金行情 |
| 交易数据 | 不接入任何交易系统 |
| 个人财务数据 | 不收集、不分析个人财务信息 |
| 投资偏好数据 | 不收集、不分析投资偏好 |

## 产品边界设计

### 系统层面的边界控制

1. **知识库边界**
   - 政策知识库仅收录政府发布的产业政策文件
   - 不收录任何金融产品、投资产品相关信息
   - 不收录证券研究报告或投资分析

2. **匹配引擎边界**
   - 匹配逻辑仅基于政策适用条件和企业特征
   - 不涉及任何财务指标的投资价值判断
   - 不涉及企业估值的投资价值评估

3. **问答系统边界**
   - PolicyCopilot 仅回答政策相关问题
   - 对于金融投资类问题，系统会明确回复"此问题超出我们的服务范围"
   - 不提供任何投资建议或收益预测

4. **报告生成边界**
   - 报告仅包含政策匹配结果和申报材料信息
   - 不包含任何投资价值分析或收益预测
   - 每份报告都附带免责声明

### 免责声明模板

> **免责声明**：本报告/回答由 YanCe Policy Agent 系统自动生成，仅供参考。本报告不构成任何政策申报成功承诺，不构成任何投资建议或金融咨询。政策申请结果取决于政府部门的审核，受多种因素影响。如需专业建议，请咨询相关政府机构或专业顾问。

## 合规承诺

### 上海衍策引擎人工智能科技有限公司承诺：

1. **不涉及金融业务**：公司主营业务为产业园区政策服务，不涉及任何金融投资、证券研究、资产管理业务。

2. **不持有金融牌照需求**：公司当前业务不需要证券从业资格、基金从业资格、投资顾问资格等金融牌照。

3. **不做收益承诺**：公司不向任何客户或用户承诺政策申请成功、补贴金额或任何投资收益。

4. **数据合规**：公司严格遵守国家数据安全法、个人信息保护法等相关法律法规。

5. **业务透明**：公司的业务范围和产品功能对合作伙伴、投资人和监管部门完全透明。

## 常见场景辨析

### 场景一：企业问"这项补贴对我们公司估值有影响吗？"

**正确回应**：
> "我们的系统专注于帮助企业了解政策信息和准备申报材料，不提供企业估值或投资分析服务。如需了解企业估值相关问题，建议咨询专业的投资顾问或评估机构。"

### 场景二：园区问"哪些行业的企业投资回报率最高？"

**正确回应**：
> "我们的系统可以帮助您分析园区内各行业的政策匹配情况，但不提供投资回报率分析。我们可以告诉您哪些行业的政策资源最丰富、匹配度最高，供您在招商时参考。"

### 场景三：用户问"应该买什么股票/基金？"

**正确回应**：
> "此问题超出 YanCe Policy Agent 的服务范围。我们专注于产业园区政策服务，不提供任何投资建议。如需投资咨询，请联系持牌金融机构。"

### 场景四：企业问"申请到这笔补贴后，资金应该怎么使用？"

**正确回应**：
> "补贴政策通常有明确的资金使用规定，我们会在材料清单中列出相关要求。但具体的资金使用规划属于企业财务决策，建议咨询专业财务顾问。"

## 边界检查清单

在产品开发和运营中，定期检查以下事项：

- [ ] 知识库中是否混入了非政策类内容
- [ ] 匹配结果中是否出现了投资价值判断
- [ ] 问答系统是否会回答金融投资类问题
- [ ] 报告中是否包含了收益预测或承诺
- [ ] 市场宣传材料中是否使用了金融相关术语
- [ ] 客户沟通中是否清晰传达了业务边界
- [ ] 免责声明是否在所有输出中正确展示
