## 海报一：品牌主视觉

### 文案

| 项目 | 内容 |
|------|------|
| **中文标题** | 让惠企政策精准触达每一家企业 |
| **中文副标题** | YanCe Policy Agent — 园区公司的 AI 政策服务 Agent |
| **English Tagline** | Every Policy. Every Enterprise. Perfectly Matched. |

### 视觉描述

以深蓝色科技背景为底，中央是一个发光的数据网络节点图，节点连接着不同的政策文件图标和企业图标。底部有上海城市天际线的剪影。整体感觉科技、专业、可信赖。

### 图片生成提示

```
A futuristic digital network visualization with glowing blue nodes connected by light beams, 
representing policy documents and company profiles being intelligently matched. 
Deep navy blue background with subtle circuit board patterns. 
At the bottom, a minimal silhouette of Shanghai skyline. 
Clean, professional, tech-forward aesthetic. Corporate presentation style.
No text, no watermarks.
```

### 规格

| 用途 | 尺寸 |
|------|------|
| 路演主屏 | 1920x1080 (16:9) |
| 易拉宝 | 80cm x 200cm |
| 社交媒体封面 | 1200x630 |

### 使用场景

- 路演主视觉背景
- 展位易拉宝
- 官方网站 Hero 区域
- 社交媒体品牌宣传

## 海报二：痛点场景

### 文案

| 项目 | 内容 |
|------|------|
| **中文标题** | 政策那么多，你的企业匹配到了吗？ |
| **中文副标题** | 园区政策服务的三大痛点，Policy Agent 一次解决 |
| **English Tagline** | Hundreds of Policies. Thousands of Companies. One Agent. |

### 视觉描述

画面分为左右两部分。左侧是杂乱的政策文件堆积、工作人员焦头烂额的抽象表现（用扭曲的文档图标和问号表示）。右侧是整齐有序的数据流和匹配结果（用排列整齐的卡片和连线表示）。中间有一条光线分割，象征从混乱到秩序的转变。

### 图片生成提示

```
Split composition: Left side shows chaotic pile of floating document icons, question marks, 
and overwhelmed abstract figure silhouettes in muted gray tones. 
Right side shows organized, flowing data streams with neatly arranged card UI elements 
connected by clean lines in vibrant blue tones. 
A bright light beam divides the two halves. 
Modern, editorial illustration style. Clean and professional.
No text, no watermarks.
```

### 规格

| 用途 | 尺寸 |
|------|------|
| 路演第二页背景 | 1920x1080 (16:9) |
| 微信公众号配图 | 900x500 |
| 宣传册内页 | A4 竖版 |

### 使用场景

- 路演痛点页面背景
- 社交媒体问题引入宣传
- 宣传册内页配图

## 海报三：产品功能

### 文案

| 项目 | 内容 |
|------|------|
| **中文标题** | 五大模块，覆盖政策服务全链路 |
| **中文副标题** | 政策问答 / 企业画像 / 智能匹配 / 材料生成 / 服务工单 |
| **English Tagline** | Five Modules. One Complete Policy Service Pipeline. |

### 视觉描述

五个圆形或六边形图标围绕中心排列，每个图标代表一个模块（对话气泡=PolicyCopilot、人物轮廓=CompanyProfile、连线网络=SubsidyMatch、文档= BriefMaker、工作台=ParkOps）。图标之间用光线连接，中心是 YanCe 的 Logo。整体感觉模块化、系统化。

### 图片生成提示

```
Five hexagonal icons arranged in a circular pattern around a central glowing core. 
Each icon represents a different function: chat bubble for Q&A, person silhouette for profiling, 
connected nodes for matching, document for report generation, dashboard for operations. 
Icons connected by luminous blue lines forming a network. 
Dark gradient background from deep blue to purple. 
Isometric tech illustration style, clean and modern.
No text, no watermarks.
```

### 规格

| 用途 | 尺寸 |
|------|------|
| 路演模块页背景 | 1920x1080 (16:9) |
| 产品介绍页 | 1200x800 |
| 宣传单页 | A5 横版 |

### 使用场景

- 路演产品模块页面
- 产品官网功能介绍
- 宣传册功能页

No text, no watermarks.

| 主色-科技蓝 | 深蓝 | #1A3A6B |

| 中性-浅灰 | 浅灰 | #F5F6FA |
| 信任-白色 | 白色 | #FFFFFF |

### 字体规范

| 用途 | 中文 | 英文 |
|------|------|------|
| 标题 | 思源黑体 Bold | Inter Bold |
| 正文 | 思源黑体 Regular | Inter Regular |
| 数据 | DIN Alternate | SF Mono |

### Logo使用

- Logo 始终位于海报右下角或右上角
- Logo 周围保留至少 Logo 宽度一半的留白
- 深色背景使用白色版 Logo，浅色背景使用彩色版 Logo
