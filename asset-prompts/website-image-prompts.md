10 image generation prompts for website sections including hero images, dashboard mockups, workflow diagrams, team environments, and park settings.

## Image 1 — Hero Section: Policy Intelligence Visualization

**Section:** Homepage hero banner
**Dimensions:** 1920 x 1080 px (16:9)

**Prompt:**
```
A stunning hero image for a tech company website. A clean, modern visualization showing
an abstract network of policy documents (represented as clean white cards) connected by
glowing teal data streams to building icons (representing park enterprises). The scene
floats in a deep navy blue space with subtle particle effects. Soft ambient lighting
creates depth. The composition is asymmetrical, with the visualization concentrated on
the right side, leaving the left side open for text overlay. Style: modern 3D render
with flat design elements, tech-forward aesthetic. Colors: deep navy (#1B3A5C), teal
(#2CB5A0), white, with subtle gold (#F59E0B) accent nodes. No people, no text, no robots.
--ar 16:9 --style raw --v 6
```

## Usage Notes

- All prompts optimized for Midjourney v6 / DALL-E 3 syntax
- For DALL-E, remove `--ar`, `--style`, `--v` parameters and specify dimensions separately
- Test each image at the target display size before final use
- Consider adding text overlays in post-production (Figma/Canva) rather than in the prompt
- Maintain consistent color palette across all images for brand cohesion
