# YanCe Policy Agent - Chrome Extension

yance.ai 园区政策分析助手 Chrome 浏览器扩展，帮助园区工作人员在浏览政策网页时，一键提取政策要点并生成企业服务建议。

## 功能特性

- **政策页面自动分析** — 打开任意政策网页，点击按钮即可自动检测并分析政策内容
- **政府网站识别** — 自动识别政府官网页面（gov.cn 等），智能判断政策类内容
- **结构化结果展示** — 政策名称、发文机关、政策摘要、适用区域、目标产业、补贴金额、申报截止日期、适用企业类型、申报条件、材料要求等
- **风险提示** — 自动识别政策申报中的注意事项和潜在风险
- **企业服务建议** — 为园区运营人员生成针对园区企业的服务行动建议
- **复制到 ParkOps** — 一键将分析结果格式化为 ParkOps 服务工单，复制到剪贴板
- **打开 Dashboard** — 一键跳转到 yance.ai 管理平台
- **政策库管理** — 一键将分析结果标记为加入园区政策库
- **下一步行动** — 清晰的操作步骤指引

## 安装方法

### 开发者模式加载（推荐）

1. 下载或克隆本仓库到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」开关
4. 点击「加载已解压的扩展程序」
5. 选择 `chrome-extension` 目录
6. 扩展图标将出现在浏览器工具栏

> 注意：当前版本需要自行提供 `icons/` 目录下的图标文件（icon16.png、icon48.png、icon128.png）。

## 使用方法

1. 在浏览器中打开任意政策网页（如政府官网政策文件、通知公告等）
2. 点击浏览器工具栏中的 YanCe Policy Agent 图标
3. 在弹出窗口中点击「分析当前政策页面」按钮
4. 等待分析完成，查看结构化分析结果（含政策名称、发文机关、补贴金额、申报截止日期等）
5. 如需生成服务备注，点击「生成园区服务备注」按钮
6. 点击「复制到 ParkOps」将分析结果复制为服务工单格式
7. 点击「打开 Dashboard」跳转到 yance.ai 管理平台
8. 如需将政策加入政策库，开启对应的开关

## 技术说明

- **Manifest V3** — 使用最新的 Chrome 扩展标准
- **纯前端实现** — 无需后端服务，本地运行
- **Permissions** — 仅请求 `activeTab` 和 `scripting` 权限，最小化权限范围
- **智能检测** — 自动识别政府网站 URL 模式和政策关键词
- **结构化提取** — 提取政策名称、发文机关、目标产业、补贴金额、申报截止日期等
